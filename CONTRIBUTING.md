# Guia de Contribuição

Obrigado pelo interesse em contribuir para este portfólio de Quality Assurance (QA)!

## Como contribuir

- Leia os projetos e documentações disponíveis.
- Abra issues para reportar bugs ou sugerir melhorias.
- Envie pull requests claros e bem documentados.
- Mantenha o respeito e a cordialidade em todas as interações.

## Requisitos para pull requests

- Siga o padrão de código e organização já existente.
- Explique claramente as mudanças no seu pull request.
- Atualize documentação, se necessário.
